# 🏛 Каталог Музейных Экспонатов

WPF-приложение для управления музейной коллекцией с разграничением прав доступа.

## 🔐 Роли и права

| Роль | Возможности |
|------|------------|
| **Пользователь** | Просмотр каталога |
| **Менеджер** | Добавление, редактирование, удаление экспонатов |
| **Администратор** | Все функции + управление пользователями и аналитика |

## 💻 Технологии

- C# / WPF (.NET Framework)
- SQL Server
- ADO.NET

## 🚀 Запуск

1. Настройте строку подключения в `DatabaseHelper.cs`
2. Откройте решение в Visual Studio
3. Нажмите F5

##  Тестовые учётные записи

| Логин | Пароль | Роль |
|-------|--------|------|
| admin | admin123 | Администратор |
| manager | manager123 | Менеджер |
| user | user123 | Пользователь |

## 📋 Функционал

- ✅ Авторизация и регистрация
- ✅ Каталог с поиском и фильтрацией
- ✅ CRUD-операции (для менеджеров и выше)
- ✅ Админ-панель:
  - Управление пользователями (смена роли, блокировка)
  - Системная аналитика (статистика, топ экспонатов)
- ✅ Кнопка выхода

## 🏗 Структура БД

Основные таблицы:
- `Users` — пользователи
- `Roles` — роли (Admin, Manager, User)
- `Exhibitss` — экспонаты
- `HallsStorages` — залы
- `Expositions` — экспозиции
